# Garment Viewer

웹 기반 3D 의복 뷰어. CLO 3D 등에서 만든 의복 모델을 브라우저에서 보고, 색상·원단·배경을 바꿔보고, 휴대폰에서는 AR로 실제 공간에 배치해 볼 수 있다.

🔗 **데모:** https://chanwooryu-create.github.io/GarmentViewer-Site_RYU/

## 기능
- 의복 선택 (Hoodie / Hoodie+Slacks / Jumper) 및 내 파일 불러오기 (.glb / .fbx / .obj)
- 회전·줌·이동 (마우스 / 터치), 의복 크기에 맞춘 자동 시점
- 옷 색상 변경, 원단 텍스처 교체 / 크기 / 회전, 배경 교체
- 새 옷·파일을 불러오면 재질 상태 자동 초기화
- 화면 구성 공유 링크, 스크린샷 저장
- AR로 보기 (모바일)

## 개선점
강의의 Garment Viewer를 기반으로 다음을 개선했다.

**화면·조작**
- 반응형 화면 — 창/기기 크기에 맞춰 자동 조정 (기존: 고정 1000×600)
- OrbitControls 도입 — 상하·좌우 회전·팬·줌·관성, 터치 지원 (기존: 좌우 1축, 마우스 전용)
- 의복 크기에 맞춘 카메라 자동 프레이밍
- 로딩 진행률 표시

**재질·스타일링**
- 옷 색상 변경(컬러 피커)
- 원단 텍스처 교체 + 크기·회전을 슬라이더·수치 입력으로 실시간 조절(양방향 동기화)
- 접이식 메뉴(Garment / Material)로 UI 정리
- 환경맵 기반 조명(은은한 반사, 천 느낌에 맞춘 광택 억제)
- 새 옷·파일 로드 시 재질 상태 자동 초기화

**파일·연동·배포**
- 내 파일 불러오기 — .glb / .fbx / .obj 드래그&드롭 (CLO 추출물 등)
- 화면 구성 공유 링크(URL)
- AR로 보기 (모바일)
- GitHub Pages 정적 호스팅으로 외부 접속 가능

## 배포 및 구현 노트
외부에서도 접속할 수 있도록 GitHub Pages로 호스팅했다. GitHub Pages는 정적 파일만 제공하는 환경이라 서버사이드 스크립트(ASP 등)나 데이터베이스를 사용할 수 없다.

그래서 데이터베이스로 의복·원단·배경 목록을 관리하는 대신, 자바스크립트 안의 목록으로 구성하고, 사용자가 자신의 모델 파일(.glb 등)을 직접 불러올 수 있게 했다. 자산 수가 적고 고정적이라 이 방식으로 충분하며, 별도 서버 없이 배포·운영할 수 있다.
